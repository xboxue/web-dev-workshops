### Core Concept Review

- component
  - building blocks of React
  - combines logic (JS) and presentation (JSX)
- props
  - data passed to a component
  - immutable
- state
  - internal data specific to component
  - data that changes over time

### Components

```jsx
const Hello = props => {
  return (
    <div>
      <h1>Hello World!</h1>
    </div>
  );
};
```

---

map, filter, reduce

- Returns new array

```jsx
arr = [1, 2, 3];

arr.map(x => x * 2); // [2, 4, 6]
arr.filter(x => x > 1); // [2, 3]
arr.reduce((sum, x) => sum + x, 0); // 6
```

### Lists

- Display lists of items with map

```jsx
const List = () => {
  const arr = [1, 2, 3];
  return (
    <ul>
      {arr.map(x => (
        <li>{x}</li>
      ))}
    </ul>
  );
};
```

### Keys

- List elements need unique keys
- React needs to identify if list changes
- Try not to use index

```jsx
return (
  {arr.map(x => (
    <li key={x}>{x}</li>
  ))}
);
```

### Forms

```jsx
const handleChange = e => {
  setValue(e.target.value);
```
