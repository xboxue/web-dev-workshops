### Block and Inline Elements

<Box width={1/2}>

- **Block elements** start on a new line and take up full width of parent
- **Inline elements** don't start on a new line and only take up width of content

</Box>

---

```jsx
<h2>Headings are block elements</h2>
<p>So are paragraphs. <em>These</em> are <strong>inline</strong></p>
```
<h2>Headings are block elements</h2>
<p>So are paragraphs. <em>These</em> are <strong>inline</strong></p>

### Box Model

- Every HTML element is a box

<Box bg="#dedede" px={5} pb={5} pt={4} mt={4} fontSize={5}>
  Margin
  <Box bg="papayawhip" px={4} pb={4} pt={1}>
    Border
    <Box bg="lightblue" px={5} pb={5} pt={4}>
      Padding
      <Box bg="white" p={6}>
        Content
      </Box>
    </Box>
  </Box>
</Box>

---

### Box Properties

- **Content** - The text, image, or media content
- **Padding** - The space between the box's content and border
- **Border** - Line that goes around content and padding
- **Margin** - The space between the box and other boxes

---

### Padding

```jsx
div {
  padding-top: 20px;
  padding-bottom: 20px;
  padding-left: 40px;
  padding-right: 40px;
}
```
<Box px={40} py={20} bg="lightblue">
  Content
</Box>

---

### Shorthand Property 

```jsx
  // Vertical Horizontal
  padding: 20px 40px;

  // Top Right Bottom Left
  padding: 20px 40px 20px 40px;
```

<Box px={40} py={20} bg="lightblue">
  Content
</Box>

---

### Border

---

### Margin vs Padding
- Margins are always transparent
- Vertical margins collapse
- Can't click on margin
- Inline elements ignore top and bottom margin

---

### Aligning Boxes

```jsx
div {
  margin: auto; // center
  margin-left: auto: // right-align
  margin-right: auto: // left-align
}
```

### Flex Container

```jsx

  display: flex;
}
```

---

### Flex Container Properties

- flex-direction - direction of items
- justify-content - alignment of items along main axis
- flex-wrap - wrap items to new line

---

```jsx
div {
  display: flex;
  // row | column | row-reverse | column-reverse
  flex-direction: column; 
  flex-direction: column-reverse; 
}
```
